# 拷问.skill 演化记录

> 本文件记录拷问.skill的形成过程，供未来Agent理解"为什么这八个角度"而非其他。

## 缘起（2026-05-27）

用户在飞书分享了一篇用「得到大脑·拷问一下」功能生成的笔记《为什么你越努力，反而越焦虑？》。对比此前我撰写的「毛选思维专栏」同主题文章，发现得到大脑版虽然篇幅不长，但通过系统性"拷问"，在深度和实操性上显著更优。

## 解剖：得到大脑"拷问一下"的7个底层操作

通过逐层拆解得到大脑版对原笔记的操作，提取出7个原子动作：

| 操作 | 具体表现 |
|:---|:---|
| 挑战舒适判断 | "这句话只说对了一半" |
| 引入关键区分 | "分不清'快到了'和'走错了'" |
| 语境还原 | "这句话的语境是战略防御中的坚持" |
| 概念祛魅 | "坚持本身不产生有利局面" |
| 重构问题 | "真正的问题不是'该不该坚持'，而是……" |
| 提供框架 | 三信号判断体系 |
| 设定边界 | 方向不对→放弃不是失败是止损 |

关键洞见：本质不是润色，而是**压力测试+分层深挖**。

## 从7个操作泛化为8个角度

7个操作中，"挑战舒适判断"和"重构问题"本质都是对一个论断的质疑，但在不同层面上运作：
- "挑战舒适判断"是对已有结论的质疑 → 前提拷问
- "重构问题"是提升问题层次 → 深挖拷问

因此拆为两个独立角度。同时补入缺失的维度：
- 反例拷问（得到大脑版未显式使用，但逻辑完备性需要）
- 实操拷问（从"读者能不能做"出发的交付检验）

最终形成8个角度，按压力等级排布：
- **等级1（必过）**：前提、边界、语境
- **等级2（深度）**：区分、反例、深挖
- **等级3（交付）**：实操、反面

## 验证

用8个角度回头拷问我们之前写的毛选专栏文章《为什么你越努力越焦虑？》，得出综合评级：🟡修复（需补充反例、边界说明和具体抓法）。
